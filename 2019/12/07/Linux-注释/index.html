Linux 注释

Linux
date –help 查看信息
man -k command 查看帮助

目录板块：

rmdir -p 子目录删除后如果其父目录为空，将其父目录也删除,，使用时不能在父目录中，不然只会删除需要删除的目录，不会删除父目录
rmdir 目录名 可直接删除（为空时）
mkdir -p 创建的目录（name） 创建完整的目录结构，可指定目录的下级
mkdir -m 设置权限

文件板块

touch 建立文件

cp [option] 被复制的文件 复制的文件路径[及名字] 不希望重新命名是，可以直接写路径

  • -i 在覆盖文件是给出提示 （只是覆盖文件内容）
  • -f 不询问直接创建
  • -r 若给出的源文件是一个目录文件，则将其目录下的所有文件复制到目标文件地址下
mv命令，将文件或者目录改名或将文件从一个目录移到另一个目录
  • mv -i 交互式覆盖文件
  • mv -f 非交互
  • mv 更改前文件名 更改后文件名 重命名文件名
more 文件名 显示文档，按空格表示翻行,只能往下翻
  • -p 清除屏幕后显示内容
  • -d 提示用户按q退出
  • -f 计算原文本行数
  • -s 将连续两行以上的空行替代为一行

less 文件名 显示文档，按空格表示翻行,b往上翻

file 文件名
辨识文件类型
-z 尝试解读压缩文件的内容

sort 文件名

对文件内容进行排序
-c 判断是否是排好序， 没排好就打印错误信息
cat -A 如果含特殊字符，显示出来
把text1和text2文件中的内容放大text3中
cat text1 text2 > text3

find [option] filename

其中，寻找条件可以用逻辑运算符组成 -a(and) -o(or) !(not)
-name ‘字串’ 查找文件名匹配所给字串的所有文件，可用*等通配符
-lname 链接文件
-gid n 查找属于id号为n的用户组的所有文件
-uid n 属于用户
-group string 查找属于用户组为所给字符串的用户组的所有文件
-user string 用户名
-empty 查找大小为0
-path string 路径

grep [option] [search pattern] [file1,file2,…]

-b 会显示出现关键字的位置及该行内容
-c 显示匹配行的数量
-n 显示满足条件的行的在原文本的行号 首行为1
-v 输出不包含指定模式的所有行
-l 显示首次匹配串所在的文件名并用换行符隔开，一个文件只出现一次

用户，用户组板块

useradd +名字

-g 指定用户组
userdel -r +名字 删除用户 一般用-r表示同时删除用户的主目录
usermod -g 修改用户所有组
groupadd
groupmod -o 修改用户组名字

chown 用户名 文件名 改变文件的所有者

改变目录的所属用户，对目录下的文件没有影响，但是若想全部改变 可用-R参数

chgrp 用户组 文件名 改变文件所有组ps

参数-R表示全改

chmod 改变访问权限(读，写，可执行)

例 chmod 777 a.txt

lsattr 查看文件属性

-R 递归显示所有文件属性

cut option 【file】 截取（都会自动截取每行）会改变文件

-c 以字符为单位截取
-b 字节
-f 域 通常和-d放在一起用， -d设置默认间隔符，-f设置域的位置
date | cut -c 15-19

comm文本比较 第一列显示第一个文本中独有的行 第二列显示第二个文本中独有的行 第三列显示共有的行
nl filename 为文本创建行号
awk 文本分析工具

awk ‘/匹配的内容/ {需要的命令}’ file
awk ‘{print $1} ‘ one 输出第一列
$0 所有列
print NR 输出行数
NF 显示所有列数 一个字符串算一列
-F 指定间隔符
awk -F: ‘{print $1} ‘ one

#!/bin/bash

进程

ps 显示进程

只输ps，只会显示当前用户正在运行的进程信息，不会显示进程状态
PID（进程ID） TTY（控制终端名称）TIME（消耗CPU总时间和） CMD（正在被执行的命令的名称）
j 以任务格式显示进程状态
ax 显示所有进程
e 显示所有进程 包括守护进程
aux 显示所有用户进程
kill命令 终止进程

创建变量 用= 两边不能有空格
获取变量 加$符号
当$获取的是没创建的，会自动创建，并赋值为null

#注释
用expert可以将局部变量导出变成环境变量
子进程对变量的修改不会传到父进程中
为了区分变量和紧跟在变量后的值，会把变量用{}括起来 $(file)1

shell 输出

echo 输出字符串 -n表示输出不换行！！
printf format [arguments] 不会自动换行
printf “%s\n” “hello world”
-左对齐 有格式符

shell 输入

read 运行之后在终端输入
-t time 在t秒内结束输入
-p ‘String’ 使用String作为输入提示
-e 读入一行
可以同时读取多个变量

shell分支控制语句

if express；then
。。。
elif expressed； then
。 。 。
else expressed； then
。。。
fi
（注意分号和格式）
case value in
expression ) command
;;
esac
(注意 ；；和 ）)

循环控制语句

while 【expression】；do
command
done
(注意；和do done)
until 【expression】；do
command
done
表达式为假是才执行

for循化

for（（1；2 ；3 ；））； do
command
done
for var

test命令：

=~ 左边字符串匹配右边正则表达式 返回true

判断整数的表达式：

num1 -eq num2 等于
num1 -ne num2 不等于
num1 -le num2 小于等于
num1 -lt num2 小于
num1 -ge num2 大于等于
num1 -gt num2 大于

字符串：

str 为空
-n str 长度大于0
-z str 长度等于0
str1 == str2 等于
str1 != str2 不等于

[ ] && [ ]
([ ]||[ ])

参数

获取位置参数

$0 执行程序所在的完整路径
$# 参数的数目
$ 所有参数列表 用“$” 会把所有参数以空格分开当做字符串
$@ 所有参数列表 区分没一个参数 “$@”也会单独区分每一个参数
$? 最后一次运行返回的值 相当于返回值

数组：

unset array 删除数组
unset array[1] 删除数组中值为1的数
array={1 2 3 4 5 6}
array={[0]=1 [3]=2}
array+={4 6} 在数组后面追加4和6
array[2]=10 定义数组，但是只会为下标为2的数组分配空间 0 1不会
echo ${array[2]} 获取数组下标为2是值

函数：

funcname（）{
command
}
funcname 参数1 参数2 …
返回值为$?

统计字符 wc -c
单词 -w
