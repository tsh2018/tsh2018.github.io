Linux 注释

~~ Linux ~~
date –help 查看信息
man -k command 查看帮助

##* 目录板块：
rmdir -p 子目录删除后如果其父目录为空，将其父目录也删除,，使用时不能在父目录中，不然只会删除需要删除的目录，不会删除父目录
rmdir 目录名 可直接删除（为空时）

mkdir -p 创建的目录（name） 创建完整的目录结构，可指定目录的下级
mkdir -m 设置权限

##* 文件板块

touch 建立文件
cp [option] 被复制的文件 复制的文件路径[及名字] 不希望重新命名是，可以直接写路径
-i 在覆盖文件是给出提示 （只是覆盖文件内容）
-f 不询问直接创建
-r 若给出的源文件是一个目录文件，则将其目录下的所有文件复制到目标文件地址下

mv命令，将文件或者目录改名或将文件从一个目录移到另一个目录
mv -i 交互式覆盖文件
mv -f 非交互
mv 更改前文件名 更改后文件名 重命名文件名
