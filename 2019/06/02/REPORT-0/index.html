2.总结

Vb 可以通过拖动给出的控件快速建立一个人机交互界面，再通过后台编程对各个控件进行关联和联系，可以快速创建一个可视化的简单程序，并且 VB包含了多种控件类型 也可以轻易的链接数据库 ,操作简便且结构简单，而且模块化的后台界面很容易上手,vb包含丰富的库函数也可以提高编程的简便性。
vector是一个能够存放任意类型的动态数组
