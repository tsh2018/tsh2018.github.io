2.总结

Math.random()函数， 返回【0，1】之间的随机数， 为double型；
void memset(void s, int ch, size_t n);
memset:作用是在一段内存块中填充某个给定的值，它是对较大的结构体或数组进行清零操作的一种最快方法 ；
构造器的调用必须位于第一句;
Elme Type :
泛指某一种特定数据类型，默认为int型，一般用于数据结构
system函数
是可以调用一些DOS命令,比如
system(“cls”);//清屏,等于在DOS上使用cls命令
dfs算法 是一个对连通图进行遍历的算法。它的思想是从一个被选定的点出发一条路走到底，如果得不到目的解，那就返回到上一个节点，然后换一条路继续走到底，直到找到目的解返回或者全部遍历完返回一个事先定好的值。dfs一般借用递归完成整个算法的构造。
bfs算法 是一个对连通图进行遍历的算法。它的思想是从一个被选定的点出发；然后从这个点的所有方向每次只走一步的走的到底,如果得不到目的解，那就返回事先储存的值，如果找到直接返回目的解， 多用队列等知识实现，包含内容较多，个人觉得难度较大。
目前自己对于dfs和bfs还不熟悉，应用还比较困难。
对JAVA还不够熟悉，对面向对象的理解还不够深刻
